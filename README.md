## 📌 About

**OUTRE Tailor System** is a complete production-ready web application for tailor shops to manage orders, deliveries, and customer data efficiently. Built with Laravel 13, Tailwind CSS, and AJAX for seamless user experience.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Order Management** | Create, update, delete orders with AJAX - No page reload |
| 🔍 **Live Search** | Instant search by name, phone, or serial number |
| 📅 **Reminder System** | 1-day before delivery reminders with Print/PDF/WhatsApp |
| ✅ **Auto Status** | Orders auto-complete when delivery date passes |
| 📊 **Dashboard** | Real-time statistics with active orders overview |
| 📱 **Responsive Design** | Works perfectly on mobile, tablet, and desktop |
| 👥 **Customer Management** | Track customer history and preferences |
| 💰 **Payment Tracking** | Manage advance payments and dues |
| 📋 **Order History** | Complete order history for each customer |
| 🖨️ **Print/PDF Export** | Generate invoices and reports |
